# stick_it
Final Project for Web Programming Winter 2022 made in collaboration with Angela Shanmugam and Sarim Ali Syed.

## The goal
Our mission was to develop an intuitive tool for users to keep track of all their diverse tasks to help them achieve a balanced lifestyle.

### Technologies Used
* Node.js
* JavaScript
* CSS
* HTML
* MySQL
* Handlebars

### Concepts Learnt
* Model-View-Controller architecture
* API testing
* Templating
* CRUD paradigm
